### Vrinda Store Data Analysis Dashboard
#### Project Overview
**This project is an interactive Annual Sales Report developed to analyze the performance of Vrinda Store. By processing over 30,000 rows of data, I created a dashboard that helps visualize sales trends, customer demographics, and channel effectiveness to support data-driven business decisions.**
#### data Processing Steps
- **Data Cleaning:** Handled null values and standardized entries (e.g., merging "M" and "Male" into a single category).

- **Data Transformation:** Created custom columns like Age Groups (Teenager, Adult, Senior) and Month extracted from date fields to enable time-series analysis.

- **Visualization:** Built interactive Pivot Tables and Slicers for dynamic filtering.

### Key Insights from the Analysis
**Based on the dashboard visualizations:**

- **Top Gender Segment:** Women are the primary customers, contributing to approximately 92% of total sales.

- **Best Performing States:** Maharashtra is the top-performing state, followed by Karnataka and Uttar Pradesh.

- **Demographic Focus:** The Adult age group (30-49 years) shows the highest engagement and purchasing power.

- **Sales Channels:** Amazon is the leading contributor to overall orders, followed by Flipkart and Myntra.

- **Order Fulfillment:** More than 90% of orders are successfully delivered, while approximately 3% are returned or cancelled
